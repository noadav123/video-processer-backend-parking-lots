# video-processer-backend-parking-lots
this project is to demostrate finding a parking spot in a parking using image processing in real time, for 3 videos.
הפרויקט כרגע לא רץ עם הSTART SYETEM בגלל בעיה בmultiprocessing.
שתי הסרטונים הראשונים שלא עובדים עם יולו רצים כרגיל, בשביל לראות את המימוש של היולו בLOT3 יש להריץ את main3.1 אינדבידואלית
